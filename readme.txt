== Descripción ==

México destinos es una agencia de empresas, encargada de vender paquetes vacacionales a personas.
México Abdsense sin duda es un plugin capaz de ofrecer una cantidad de servicios que coincidan con el
titulo de la página en donde se encuentre.


== Instalación ==

1.-Descargar el plugin e instalar
2.-Activar el plugin
3.- Dirigirse a la sección de Apariencia/Widgets y colocar el widget en el lugar en donde quieren que se muestre.
